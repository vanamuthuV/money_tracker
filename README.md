# Money Tracker

Money Tracker is a Flutter-based mobile application built as an experimental project to explore AI-assisted (“vibe-coded”) development in the mobile app ecosystem.

This project was created to understand rapid prototyping, development speed, and workflow changes when building applications with AI support.

## Overview

Money Tracker allows users to:

* Create and manage expense categories
* Add and update debit and credit transactions
* View basic spending analysis
* Store financial data locally using a JSON-based storage system
* Import and export stored data manually

The application is intentionally lightweight and focused on core functionality.

---

## Tech Stack

* Flutter
* Local JSON file storage for persistence
* Android APK build

All data is stored locally in JSON format, making it portable and easy to back up or migrate.

## Purpose of This Project

This application was built to:

* Experiment with AI-assisted development workflows
* Explore rapid mobile app prototyping using Flutter
* Study how AI tools influence application structure and productivity
* Understand the evolution of development patterns in the AI era

It serves as a practical experiment in building applications quickly while observing the strengths and limitations of AI-supported coding.
